# Create repository

1. Install Klave sdk

```bash
yarn create trustless-app
```

2. Follow-up instructions
![image-title-here](/assets/images/yarn-create.png)

3. Go into the newly directory named after your trustless-app

```bash
yarn install
```

4. Open your favorite IDE
